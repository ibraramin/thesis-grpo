## Key Findings From Validation Run

### SFT is essential
- Base model: 0.2% solve, 3.0% format — useless without SFT
- SFT model (5K×1): 6.6% solve, 24.8% format — 33× better, 8× better
- SFT model (10K×2): ~15-20% solve (projected), ~35-45% format (projected)

### Baseline = zero reward
- λ_format=0 → all completions score 0.0 → σ(R)=0 → advantage=0
- No gradient signal, no learning — mathematically expected

### Cohort D outperforms baseline on all benchmarks
- SVAMP: +2.7pp accuracy, +2.3pp format
- GSM8k: +0.5pp accuracy, +1.1pp format
- MATH-500: no accuracy delta (too hard), +1.1pp format
- γ=0.01 was too fast — format decayed before model learned formatting
- Current γ=0.005

### Cohort C (graduated) chosen alongside D
- Partial credit (1.0/0.5/0.2/-0.2) provides denser reward than binary
- Orthogonal to D: tests "reward density" vs "reward dynamics"
